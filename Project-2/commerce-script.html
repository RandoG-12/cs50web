Commerce Project Script

Complete the implementation of your auction site. You must fulfill the following requirements:

  1. Models......[show admin site]
    • At least 3 models: listings, bids, and comments
  2. Create Listing (Logged in)
    • Required fields: title, description, starting bid.
    • Optional fields: image URL, category
  3. Active Listings Page (all visitors)
    • Default route showing at least: title, description, current price, photo if present
  4. Listing Page (how to see inactive listings????)
    • (all visitors): Each listing has link to full details page
    • (Logged in): add/remove listing from their “Watchlist.”
    • (Logged in): Bid on the item. Bid >= starting bid, Bid > other bids, otherwise error.
    • (Logged in, listing creator): “close” the auction, highest bidder wins, auction becomes inactive
    • (Logged in, listing winner): listing shows they won.
    • (Logged in): add comments to listing page. Page shows all comments for listing.
  5. Watchlist
    • (Logged in): visit Watchlist, each watched item has link to listing
  6. Categories
    • (all visitors): page with all listing categories, links to active listings in that category
  7. Django Admin Interface
    • view, add, edit, and delete any listings, comments, and bids made on the site.
